Satri Animoog Guide

Leveraging MIDI with Animoog

MIDI Expansion Pack

To be able to use MIDI with Animoog, you'll need to purchase the x$ 'MIDI Expansion Pack' from the Animoog Store ('STORE' tab)

MIDI Use Cases

Main usages for MIDI with Animoog:

  • Play Animoog from an external keyboard / input device
  • Record an Animoog performance in MIDI for edition and playback

MIDI Setup

MIDI setup in Animoog is done in the 'SETUP' tab
